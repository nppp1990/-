2、综合应用题第一道大题是个离散数学的证明题，应该是用数学归纳法证明的，但是做不出来，我就照猫画虎叶结点、叶结点上层成立，假设k高度成立，“证明”k+1高度的成立了。
题目是：假设二叉树T为正规二叉树，即结点分支只能为0或2，设N（T）为非叶子结点的结点数，h（T）为二叉树高度，若只有一个结点时h=0。设有函数F（T），若T只有一个结点，则F（T）=0，否则F(T) = F(T左) + F(T右) + min { h(T左), h(T右) }。
证明：F(T) = N(T) - h(T)。


第二道大题是比较常规的题型，但是鄙人不才，通过邻接矩阵画出来的无向图歪歪扭扭的，捋不直，希望老师不要扣分。然后让你求V0结点开始到各个节点的最短路径（单源最短路径问题），显然是用Dijkstra，每次加入一个集合元素，更新Dist实现。

第三道大题是一道堆排序的题将数组0123456789初始建成大顶堆，然后在HeapSort过程中依次输出9、8、7、6后，堆的情况，相信各位理解堆排序手工处理的同志们都能做出来。

3、算法编程题两道大题堪称噩梦。第一道大题是给你单链表的结构体，然后用它来实现链栈，要求写出构造链栈的函数Stack ( ) ; 析构函数~Stack ( ) ; Push ( S , &Node p ) ; Pop ( ) ; 求长度Length ( S ) ; 还有一个我忘了，反正也是基本操作函数。说明自己复习走马观花，直接把这些基本函数当成直接调用的东西“拿来主义”了，这个题感觉就是阻挡我这种跨考生的，因为没有很好的代码功底，编程基础薄弱。从基本操作函数再结合链栈这种不怎么常用的数据结构，的确是考倒了一大批。
第二道大题更是玄学了，鄙人不才，题目大概是快排&*…%一段废话之后，让你写出快速排序的Partition ( int A[ ] , int n ) ; 函数的递归实现形式，当时我就在考场喵喵喵了，因为王道和严版的教材对Partition划分枢轴的过程都采用的两个并列的while循环交替实现swap，我其实联想到的是算法导论中求股市买入卖出最大收益率那个分治法的算法实现，但是没有思路，乱写的。这道大题的括号二就更狗了……让我们优化自己的Partition ( ) ; 函数，使其能达到O ( logn )的时间复杂度。
